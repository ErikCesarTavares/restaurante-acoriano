## 📂 Estrutura do Projeto

A estrutura do projeto inclui as seguintes seções:

- **Sobre o Restaurante**: Informações sobre a história e a proposta do restaurante.
- **Eventos**: Destaque para eventos especiais e promoções.
- **Cardápio**: Exibição do cardápio interativo, separado por categorias: bebidas, entradas, pratos principais e sobremesas.
- **Fale Conosco**: Formulário de contato para os usuários enviarem mensagens diretamente pelo site.
- **Carrossel de Imagens**: Apresentação elegante do ambiente do restaurante.

---

## 📋 Funcionalidades

1. **Navegação por seções**: Facilita a navegação entre as páginas de sobre, eventos, cardápio e fale conosco.
2. **Carrossel de Imagens**: Mostra imagens do ambiente do restaurante de maneira interativa e atraente.
3. **Cardápio Interativo**: Organizado por categorias, proporcionando uma experiência de navegação agradável.
4. **Formulário de Contato**: Permite aos usuários enviar mensagens e fazer perguntas diretamente pelo site.
5. **Responsividade**: O layout ajusta-se para diferentes tamanhos de tela, garantindo uma experiência de usuário otimizada em dispositivos móveis, tablets e desktops.
